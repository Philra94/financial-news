Use pre-fetched Google search results to ground claim research when they are available.

Workflow:
- Check whether `google-search.json` exists in the current research workspace.
- Treat the listed titles, snippets, and URLs as candidate sources to inspect first.
- Prefer high-quality financial and primary sources surfaced there before broad exploration.
- If the file is missing or empty, note that Google search was not configured or did not return results, then continue with the other available evidence.

Output expectations:
- Distinguish sourced facts from interpretation.
- Cite URLs explicitly in the final markdown report.
- If a search result looks low quality, say so and do not rely on it heavily.
